BELLE HOUSE 点呼システム Ver.9.2

追加機能:
- ドライバーマイページに「今月の稼働予定日数」を表示
- ドライバーマイページに「今月の点呼実施日数」を表示
- 同じ日に複数コースを担当していても1日として集計
- 同じ日に乗務前・乗務後の両方を送信しても1日として集計
- 編集権限は従来どおり管理者のみ

上書きするファイル:
- index.html
- app.js
- style.css
- manifest.json
- sw.js

確認手順:
1. ZIP内の v92 フォルダーのファイルを現在のフォルダーへ上書き
2. Live Serverを停止して再起動
3. ブラウザで Ctrl + F5
4. ドライバーでログイン
5. ホームに稼働予定日数と点呼実施日数が表示されることを確認
