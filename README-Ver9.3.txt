BELLE HOUSE 点呼システム Ver.9.3

修正内容:
- 稼働予定日数と点呼実施日数をドライバーホームから削除
- 2つの集計をドライバー用「シフト表」の中へ移動
- 前月・翌月を切り替えると、選択した月の数字へ自動更新
- シフト表の編集権限は管理者のみ
- ドライバー側は閲覧専用

上書きするファイル:
- index.html
- app.js
- style.css
- manifest.json
- sw.js

確認手順:
1. ZIP内の v93 フォルダーのファイルを現在のフォルダーへ上書き
2. Live Serverを停止して再起動
3. ブラウザで Ctrl + F5
4. ドライバーでログイン
5. 「シフト表」を開く
6. シフト表の上に稼働予定日数と点呼実施日数が表示されることを確認
7. 前月・翌月を切り替えて数字が変わることを確認
